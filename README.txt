INSTRUCTIONS

// interaction
- single click to create a sound source and drag it to size
- click and drag the circumference to resize the source
- click and drag the center of the circle to reposition its
- double click to close the sound source
- 'r' to create a random source in a random place
- 'c' to delete all the sources
- ENTER to toggle Atomic mode
- SPACE to freeze the fly

// menu
- RIGHT key to display sound list
- LEFT key to close the list
- UP and DOWN key or mouse click to navigate the list
